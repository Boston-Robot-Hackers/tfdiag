# tfdiag

## Contaxt

A cli under ros2 which will analyze the existing tfs and look for problems. We will be using a new package for the cli called cyclopts. Begin by reading the package documentation (https://cyclopts.readthedocs.io). Also look through https://docs.ros.org/en/kilted/Tutorials/Intermediate/Tf2/Introduction-To-Tf2.html
